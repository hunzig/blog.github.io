Wireless D.O.S: 客户端与ap建立连接，通过广播插入伪造的取消身份验证报文。客户端认为该报文来自AP，已链接诶客户端自行断开连接

Wireless频带阻塞干扰：攻击信号覆盖范围内，对指定频道进行阻塞干扰

Wireless数据流截获及分析：帐号密码 通信数据 voip音频流 截获，伪造AP进行数据报文截获攻击

VOIP音频流截获：正常voip语音通信及短消息被攻击者截获，伪造转录破坏通信内容，被迫断线或收到伪造信息

伪造基站攻击：（伪造ap攻击）

Wireless摄像装置入侵：破解无线wep/wpa加密，非法连接到无线摄像头，从外部窥视内部保密环境

WPA PMK Hash Table Attack: time-memory trade-off 即内存-时间平衡法，也可以翻译成 时间-内存交替运算法
制作WPA PMK Hash Table:
Cowpatty下的genpmk工具
Aircrack-ng下的airolib-ng工具

WPA-PSK/WPA2-PSK分布式破解：zerone security team内部研发项目

  • 提交wpa握手文件
  • 添加至计算任务列表
  • 查找现有hash库列表
  • 将任务分配至各分布端
  • 运算结果返回至服务器

国际上已经开展的分布式运算：free rainbow table、蛋白质、Elcomsoft

来自nvidia的gpu技术理论速率提升至原来的30-100倍
