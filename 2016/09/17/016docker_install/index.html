二、在ubuntu下安装docker

支持版本

12.04 13.04 13.10 14.04. , 64 bit

内核 3.10以上

安装前准备

  1. 内核版本： unnamed -a

  2. 检查 device mapper. （驱动文件是否存在）

Ls -l /sys/cbass/misc/device-mapper

安装Ubuntu 维护的版本

sudo apt-get install docker.io

Scource /etc/bash_completion.d/docker.io 更新配置

安装docker维护的版本

Sudo apt-get install -y curl

Curl -sSL https://get.docker.com/Ubuntu/ | sudo sh

实例应用

Sudo docker run Ubuntu echo ‘Hello World’

加入docker组

sudo group add docker

Sudo gpasswd -a username docker

Sudo service docker restart

三、windows下安装docker

boot2docker for windows 包含

  1. https://github.com/boot2docker/boot2docker

  2. 下载docker-install.exe

  3. 安装docker-install.exe

实例应用

docker run ubuntu echo ‘Hello World’

unable to find image ‘ubuntu:latest’ locally

然后会自动下载ubuntu镜像

四、mac下安装docker—boot2docker

boot2docker=轻量级linux+virtualbox+docker客户端+管理工具

系统要求

os x 10.6 snow leopard 之后的系统

下载安装

  1. https://github.com/bootsdocker/bootsdocker

  2. 下载 boot2docker-1.5.0.pkg

  3. 安装 bootsdocker-1.5.0.pkg

bootsdocker启动过程

  1. 打开命令行通道

  2. 建立$HOME/.bootsdocker 目录

  3. 创建virtualbox ISO

  4. 启动虚拟机并运行docker守护进程

进入守护进程虚拟机命令

boot2docker ssh

其他命令查看 boot2docker help

实例应用

docker run ubuntu echo ‘Hello World’
